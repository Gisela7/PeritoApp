# PeritoApp
Sistema de cadastro de peritos
Título do projeto: PeritoApp
    Status do projeto (em desenvolvimento)
    Tecnologias que serão aplicadas: Java e MySQL
    Time de desenvolvedores: Gisela Borges
    Objetivo do software: auxiliar na expedição de alvarás
    Funcionalidades do sistema: 
    Gestão de Processos e Perícias
    Cadastro de Processos
    Controle de Honorários
   Requisitos de Sistema:
   Para que o **PeritoApp** funcione corretamente na máquina do usuário, os requisitos são:
* **Sistema Operacional:** **Windows 10 ou 11** (Recomendado), macOS ou Linux.
* **Conexão com a Internet:** **Não é necessária (Opcional)**. Como o sistema utiliza persistência de dados local, todas as operações de cadastro, consultas de perícias e geração de relatórios funcionam de forma 100% offline. A internet só será necessária caso o usuário queira buscar atualizações do sistema ou acessar links externos.
* **Ambiente de Execução:** Necessário possuir o **Java Runtime Environment (JRE)** versão 17 ou superior instalado para executar o aplicativo.
* **Armazenamento:** Mínimo de **200 MB** de espaço livre em disco para a instalação do aplicativo e crescimento do banco de dados local com as informações dos processos.
